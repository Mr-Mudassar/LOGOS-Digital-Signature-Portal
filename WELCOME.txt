🚀 Quick Start (5 minutes):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Install dependencies:
   npm install

2. Set up your database:
   - Get connection string from Supabase
   - Update .env file

3. Initialize database:
   npx prisma generate
   npx prisma db push

4. Start development server:
   npm run dev

5. Visit: http://localhost:3000
